#  Chazam - Notre Projet de Chatbot

Bienvenue sur le dépôt officiel de **Chazam**, un assistant conversationnel développé en équipe.

##  Notre Vision Collective
Nous avons conçu Chazam pour explorer les possibilités de l'intelligence artificielle conversationnelle tout en appliquant une méthodologie de travail collaborative rigoureuse.

##  Notre Stack Technique
- **Langages :** HTML/CSS/JS
- **Versionnage :** Git (Branches, Pull Requests, Tags)
- **Hébergement :** GitHub Pages pour la présentation
- **Gestion de Projet :** Tableau Kanban (GitHub Issues)
