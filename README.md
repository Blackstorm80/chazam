# Chazam Chatbot
Projet de mise en situation pour le module de Gestion de Projet (L2 Informatique).

## Description
Un chatbot basé sur ChatterBot capable de répondre en français et d'effectuer des calculs mathématiques.

## Installation
- Python 3.10+ (Patch inclus pour 3.13)
- pip install chatterbot spacy
- python -m spacy download fr_core_news_sm
